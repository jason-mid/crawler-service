# crawler-service
This is a learning project focused on creating a web crawler service.

## Features
- Crawl multiple websites and extract data.
- Configurable crawling parameters.

## Technologies Used
- Python
- Scrapy

## How to Use
1. Clone the repository: `git clone https://github.com/jason-mid/crawler-service.git`
2. Navigate to the project directory: `cd crawler-service`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the crawler: `scrapy crawl myspider`
